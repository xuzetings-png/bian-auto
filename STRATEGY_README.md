# 🚀 加密货币量化交易策略 - 升级版

## 📊 当前问题分析

你之前使用的动量策略存在以下问题：
- **过度交易**：频繁开平仓导致高额手续费
- **胜率不稳定**：单纯的动量突破在震荡市场容易被假突破止损
- **风险控制不足**：缺乏市场状态过滤器

---

## 💡 我们的新策略方案

根据研究，我们实现了三种经过验证的策略：

### 1️⃣ Bollinger + RSI 反转策略
**原理**：利用布林带捕捉价格极值，结合 RSI 超买超卖信号，配合成交量确认，进行反转交易。

**入场条件**：
- 做多：价格触及布林带下轨 + RSI < 30 + 成交量放大
- 做空：价格触及布林带上轨 + RSI > 70 + 成交量放大

**适用场景**：震荡市场表现极佳

---

### 2️⃣ 趋势跟踪策略（EMA + ADX）
**原理**：使用快慢 EMA 交叉捕捉趋势，ADX 过滤器确认趋势强度，只在强趋势中交易。

**入场条件**：
- 做多：EMA 金叉（快线上穿慢线）+ ADX > 25 + 成交量放大
- 做空：EMA 死叉（快线下穿慢线）+ ADX > 25 + 成交量放大

**适用场景**：趋势市场表现最佳

---

### 3️⃣ 均值回归策略（改进版）
**原理**：计算过去 N 周期的高低点，在价格接近区间边缘且有反转迹象时入场。

**入场条件**：
- 做多：接近区间低点 + 阳线反转 + 成交量放大
- 做空：接近区间高点 + 阴线反转 + 成交量放大

## 📈 策略回测指标

我们关注这些关键指标：
- **总盈亏** - 最终收益率
- **胜率** - 盈利交易占比（目标 > 50%）
- **盈亏比** - 平均盈利 / 平均亏损（目标 > 1.5）
- **夏普比率** - 风险调整后收益（目标 > 1.0）
- **最大回撤** - 最大历史亏损（目标 < 20%）

## 💡 根据研究的交易建议

1. **不要过度杠杆** - 建议 2-3 倍杠杆，避免强平
2. **严格止损** - 每次交易只承担总资金 1-2% 的风险
3. **分散投资** - 同时交易多个不同币种
4. **市场过滤** - 只在流动性好的主流币交易
5. **策略组合** - 可以同时运行多个策略分散风险
